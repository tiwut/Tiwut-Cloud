Get Cloud

Your Files.
Your Port. No DB.

TiwutCloud is a high-performance private cloud system built purely in PHP. Designed for custom ports, maximum privacy, and ultra-fast AVIF/4K streaming.

Zero Database

Uses the filesystem and JSON for metadata. No SQL injections, no complex backups, just folders and files.

Max Speed

Direct binary streaming via PHP allows for maximum port throughput, supporting massive files without memory limits.

PWA Ready

Install it on your iPhone or Android as a native app. Works perfectly in full-screen on tablets and computers.

INITIAL SETUP

1

Configuration

Edit config.php and set your admin_password. This key protects your entire cloud.

2

Directory Permissions

Ensure the /storage and /data folders are writable. Use chmod 777 if you are on a Linux server.

3

Icon & Branding

Place your icon.ico in the root folder. TiwutCloud will automatically detect and use it as your favicon.

CLOUD USAGE

Uploading

Use the + button to upload files, or simply drag and drop files from your computer directly into the browser window.

Rich Sharing

Click the "Edit" icon on any file and select Share UI Page. This generates a professional landing page link for recipients, featuring a high-quality preview.

Previews

TiwutCloud renders images and videos in perfect quality. Use the Full Screen icon in the preview window for an immersive experience.

Share Manager

Access the Link icon in the header to manage all active shares. You can refresh the list, copy links, or revoke access instantly.
